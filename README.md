## 🎯 Описание проекта

В условиях макроэкономических шоков и высокой волатильности классические методы инвестирования (такие как теория Марковица) теряют эффективность из-за опоры на исторические данные («ошибка оценки»). 

В данном проекте реализован **гибридный подход к оптимизации портфеля**: математический каркас Mean-Variance Optimization (MVO) сохраняется, но исторический вектор ожидаемых доходностей заменяется на **предиктивный вектор, рассчитанный с помощью алгоритмов машинного обучения**. 

В качестве источника данных выступают 5 «голубых фишек» Московской биржи (MOEX). Бэктестинг проведен на кризисном периоде 2022–2025 гг.

## 🧠 Используемые алгоритмы
1. **LSTM (Long Short-Term Memory)** — рекуррентная нейронная сеть для извлечения нелинейных временных зависимостей (основная модель).
2. **XGBoost (Extreme Gradient Boosting)** — ансамбль деревьев решений (ML-бенчмарк).
3. **MVO (Mean-Variance)** на исторических данных — классический бенчмарк (теория Марковица).
4. **Equal Weight (1/N)** — наивный равновесный портфель.

---

## 📊 Основные результаты (Out-of-Sample: 2022-2025)

Эмпирическое тестирование доказало, что применение алгоритмов глубокого обучения позволяет нейросети превентивно реагировать на рыночные шоки, "сбрасывая" рисковые активы.

| Стратегия | Доходность (CAGR) | Волатильность (Risk) | Коэф. Шарпа |
| :--- | :---: | :---: | :---: |
| **LSTM-портфель** | **+ 15.4%** | 38.9% | **0.548** |
| XGBoost-портфель | + 8.1% | **31.9%** | 0.253 |
| Марковиц (Hist) | - 5.0% | 44.0% | -0.114 |
| Равновесный (1/N) | - 17.7% | 41.9% | -0.422 |

*(Классические подходы в период кризиса показали отрицательный Шарп, разрушая капитал инвестора. LSTM-сеть обеспечила стабильный рост).*
